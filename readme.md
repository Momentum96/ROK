Rise Of Kindoms Macro
======================

## Rise of kingdoms 게임의 자원 수집 매크로 개발

### 주요 기능
- 특정 조건에 따라 팝업 형태로 등장하는 이미지의 등장 여부에 따른 동작 자동화
- 사용자의 captcha 인증 필요 시 텔레그램 봇을 활용한 알림 기능 구현  

(하단 이미지 클릭 시 해당 기능을 시연한 유튜브 영상 페이지로 이동됩니다.)
